# 01 - Cloud Concepts ☁️

## Introduction

Le Cloud Computing désigne la fourniture de services informatiques (serveurs, stockage, bases de données, réseaux, logiciels) via Internet (“le cloud”).

Au lieu d’acheter et gérer des infrastructures physiques, on utilise des ressources à la demande.

---

## 1. Pourquoi le Cloud ?

Les entreprises utilisent le cloud pour :

- Réduire les coûts d’infrastructure
- Améliorer la flexibilité
- Augmenter la rapidité de déploiement
- Assurer une meilleure disponibilité

---

## 2. Types de Cloud

###  Cloud Public
Les ressources sont partagées et gérées par un fournisseur (ex : Azure, AWS).

- Faible coût
- Haute scalabilité
- Moins de contrôle

---

### Cloud Privé
Infrastructure dédiée à une seule organisation.

- Plus sécurisé
- Plus de contrôle
- Coût plus élevé

---

### Cloud Hybride
Combinaison du cloud public et privé.

- Flexibilité maximale
- Migration progressive vers le cloud
- Complexité plus élevée

---

## 3. Modèles de service Cloud

### 🔸 IaaS (Infrastructure as a Service)
Tu gères :
- OS
- Applications
- Données

Le fournisseur gère :
- Serveurs
- Réseau
- Virtualisation

Exemple : Azure Virtual Machines

---

### 🔸 PaaS (Platform as a Service)
Tu gères uniquement :
- Applications
- Données

Le fournisseur gère :
- Infrastructure
- OS
- Runtime

Exemple : Azure App Service

---

### 🔸 SaaS (Software as a Service)
Tout est géré par le fournisseur.

Tu utilises simplement l’application.

Exemples :
- Microsoft 365
- Gmail

---

## 4. Avantages du Cloud

-  Coût réduit (pay-as-you-go)
-  Rapidité de déploiement
-  Scalabilité (augmentation automatique des ressources): Horizontale ou verticale
-  Sécurité avancée
-  Accessibilité mondiale

---

## 5. Scalabilité & Élasticité

### Scalabilité
Capacité à augmenter les ressources (CPU, RAM, etc.) ou les machines

### Élasticité
Capacité à augmenter/diminuer automatiquement selon la demande.

---

## 6. Haute disponibilité

Permet de garantir que les services restent accessibles même en cas de panne.

---

## Résumé

Le cloud permet aux entreprises de consommer des ressources IT à la demande, avec plus de flexibilité, de sécurité et de performance.
